Javascript技术题

1. 请列举一些数组元素去重方法

方法一：构建结果数组；for循环中每次从原数组中取出一个元素，用这个元素循环与结果数组对比；若结果数组中没有该元素，则存到结果数组中。 方法二：构建结果数组；先将原数组进行排序；检查原数组中的第i个元素与结果数组中的最后一个元素是否相同；如果不相同，则将该元素存入结果数组中。 方法三：创建结果数组；创建一个空对象；for循环时，每次取出一个元素与对象属性进行对比，如果这个属性不存在，则把它存放到结果数组中，同时把这个元素的内容作为对象的一个新属性，值为1。

2. javascript的作用域和作用域链是什么？

JavaScript的作用域不是以花括号包围的块级作用域，而是函数作用域，在一个函数中定义的变量只对这个函数内部可见。 JavaScript函数对象拥有一个内部属性[[Scope]]，该内部属性包含了函数的作用域链，作用域链包含了它外层的变量对象。在函数执行过程中，每遇到一个变量，都会经历一次标识符解析过程以决定从哪里获取和存储数据。该过程从作用域链头部（活动对象）开始搜索，如果找到了就使用这个标识符对应的变量，如果没找到继续搜索作用域链中的下一个对象。

3. javascript变量声名提升是什么意思？

javascript的变量声明具有hoisting机制，JavaScript引擎在执行的时候，当前作用域内的声明都会提升到作用域的顶部，相同的name会根据提升的优先级被覆盖

4. 如何解决回调层次过深的问题？

使用Promise，Promise的then链式调用避免了异步函数之间的层层嵌套，将原来异步函数的嵌套关系转变为便于阅读和理解的链式步骤关系。

5. Ajax是指什么？jsonP是指什么？两者的原理分别是？

ajax 即“Asynchronous JavaScript and XML”（异步 JavaScript 和 XML），是一种用于创建快速动态网页的技术，Ajax 的核心是 JavaScript 对象 XmlHttpRequest，可以使用 JavaScript 向服务器提出请求并处理响应，而不阻塞用户。 JSONP(JSON with Padding)是JSON的一种“使用模式”，可用于解决主流浏览器的跨域数据访问的问题，利用<script>元素的这个开放策略，网页可以得到从其他来源动态产生的 JSON 资料。

2. 在canvas下如何优化文字的渲染速度？

使用链接 (chaining),对希望渲染的文本样式属性进行分组，相应地指定样式属性，然后调用闭合组的 CanvasText 对象的 append() 函数，并存储它供后续使用。渲染文本时，需要遍历这些组，然后指定样式并应用它。在迭代组时，保留之前的样式状态的工作记录，并在必要时覆盖这些记录。

4. transform和setTransform接口的区别

transform是替换绘图的当前转换矩阵，它会添加一个新的变换矩阵，再次绘制矩形 setTransform是将当前转换重置为单位矩阵。然后运行 transform()，它会重置前一个变换矩阵然后构建新的矩阵
